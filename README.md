# Fitness_DB
Relational database tracking workouts, meals, and supplements. Provides actionable insights to coach clients toward fat loss, muscle gain, and fitness goals using SQL and data-driven analysis.
**Coaching Focus:**  
- Track client workouts, including exercises, sets, reps, and intensity  
- Record meals and supplement intake to monitor nutrition adherence  
- Analyze patterns over time to optimize programming and dietary guidance  
- Identify strengths, weaknesses, and progress to help clients stay on track  

**Technical Skills / Tools:**  
- Relational database design with **one-to-many and many-to-many relationships**  
- SQL queries: `JOIN`, `GROUP BY`, `COUNT`, logical and comparison operators  
- Data integrity ensured with **foreign keys** and consistent constraints  
- Reports and queries generate actionable insights for fitness coaching decisions  

**Example Use Cases:**  
- Quickly see which clients are hitting their nutrition or workout targets  
- Identify exercises or meals correlated with client progress  
- Generate reports to review weekly or monthly performance trends  

**Goal:**  
This database demonstrates how combining **data management with practical coaching strategies** can improve client results, streamline program adjustments, and provide clear, measurable feedback.
